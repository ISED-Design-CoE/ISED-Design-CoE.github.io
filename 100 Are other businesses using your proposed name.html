Notes about this mockup:

  • This page shows up to 35 results, depending on the result of the NUANS fuzzy match algorithm
  • If the algorithm does not have any results, then skip this page and take the user directly to the next
  • Information in expanded view is the same as in the modal windows of the current integrated process (see mockup), with the following changes:
    • For federal corporations (e.g. Between Us Publishing Inc above), "Company number" has been hyperlinked to the appropriate information in the Search for a Federal Corporation
    • "Jurisdiction code" has been expanded out into "Jurisdiction" (e.g. "ON" -> "Ontario"), for readability
    • "City" has been renamed "Head office", to distinguish it from where the company is doing business
    • "Business activity (NAICS)" includes not just the NAICS code but also a human-readable name for the code, as well as a hyperlink
      • The name has also been pluralised to "Business activities (NAICS)" if there is more than one activity
    • For trademarks (e.g. Between Us Rewards Program above), "Wares and services" has been removed because it can be quite long
  • This page also has a two-name version
