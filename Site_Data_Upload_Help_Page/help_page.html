Introduction

To ensure that spectrum is used for the benefit of all Canadians, Innovation, Science and Economic Development (ISED) Canada requires spectrum licensees to provide detailed information about stations they deploy in Canada.

Read more about your obligations to submit station data - Decision on Amendments to Requirements for Spectrum Licensees to Submit Technical Information about Sites

How to upload site information

Register your web profile

Before you can upload your station data, you will need to register and create a web profile in the Spectrum Management System (SMS).

Online registration

When should I upload my site data

You can determine the due date of your next required upload using either of these methods:

Method 1: Use the Spectrum Licenced Site Data Browser
Enter the search parameters and scroll to the far right of the screen to view the last upload date. Licensees must submit data by the following month.

Method 2: Use the Spectrum Licence Site Data Status Report
The furthest right column shows when the next upload is due.

Upload file format

Your site data file must be in CSV format, and follow the format outlined at Text Format for Spectrum Licence Site Data Upload. Each row represents one station.

Each file cannot exceed 60 MB. Use the “Append mode” feature to upload multiple files.

An example CSV file is available: Sample_54_columns.csv

What to expect after uploading

Successful upload

Summary report

You will receive a summary report that acts as the receipt that your data has met the conditions of your licence.

View stations report

You will also receive a stations report of information about your stations.

Unsuccessful upload

Error report

The error report details the issues found with your site data upload. You can use the Site data upload help page to assist with addressing any issues.

If you have any questions with your site data upload, email us at spectrumlicensing-licencesdespectre@ised-isde.gc.ca
