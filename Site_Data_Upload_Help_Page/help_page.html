How to upload site data Spectrum licensees are required to provide detailed information about their sites to ensure that spectrum is used to benefit all Canadians. For more information on the site data upload requirements, visit Decision on Amendments to Requirements for Spectrum Licensees to Submit Technical Information about Sites. On this page

Create your web profile If you haven’t already, you will need to Create your Web Profile to upload your site data on the Spectrum Management System (SMS). When to upload your site data Your site data must be uploaded on a monthly basis. You can find the due date of your next required upload by either: 1) Viewing your last upload date
Using the Spectrum Licenced Site Data Browser to view your last upload date. 2) Viewing your status of your licence
Using the Spectrum Licence Site Data Status Report to check when your next upload date is due. Upload your site data Upload your site data in a .csv file format. Each row in the file is used to represent an individual antenna. For example, if you deployed a site with multiple antennas (transmitters and/or receivers), provide information about each antenna on a separate row. Each file cannot exceed more than 60 MB. If you have more sites than can fit within a 60 MB file, then you can use the append feature to add to previously uploaded files. View terrestrial format View satellite format What to expect after uploading If your upload is successful You will receive a summary report that acts as the receipt that your data has met the conditions of your licence. You will also receive a sites report with information about your sites. If your upload fails If your upload fails, you’ll receive an error report detailing the issues for you to fix. The format page found in the How to format your .csv file section, provides a breakdown of the error messages and how you can fix them. Need help? If you have any questions email us at
