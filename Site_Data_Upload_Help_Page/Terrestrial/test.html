Text format for terrestrial spectrum licence site data upload

Learn about the format, validation rules, and errors when uploading spectrum licence site data. Download the .csv file below to use as a template.

Field names

Letter Name

A

Spectrum licence number

Enter the number of your spectrum licence.

Example

10898101

Format

  • Must belong to an account in your web profile
  • Must be a granted (paid) spectrum licence, and not expired
  • Must be exactly 9 digits long including leading zeroes as necessary
  • Do not include the licence version number

Potential error(s)

The licence number does not contain a leading 0, the licence number does not belong to your account, the licence number is not in Granted status, or the licence is expired.

B

Upload reference number

Optional field to enter a reference name for your site.

Example

Monthly upload, March 2023

Format

Maximum length of 63 characters.

C

Contact name

Name of the person who may be contacted.

Example

John Smith

Format

Maximum length of 254 characters.

Potential error(s)

The Contact Name is missing or exceeds the character limit.

D

Business telephone number

Enter the telephone number of the person who may be contacted.

Example

613 – 174 – 5555 x 993

Format

  • Minimum length of 10 characters
  • Maximum length of 254 characters

Potential error(s)

The Business Telephone Number is missing or exceeds the character limit.

E

Email address

Enter the email address of the person who may be contacted.

Example

john.smith@mail.com

Format

  • Maximum length 254 characters
  • For multiplate email addresses, separate them with commas

Potential error(s)

The E-mail Address is missing or exceeds the character limit.

F

Station location

Enter the station is located or specify if there are no changes or no stations.

Example

WR0123 - Strandherd Repeater Site

Format

If you currently have a station
If you currently have a station deployed, enter the location of your station (Maximum length of 63 characters).

If you have no changes to stations
If you have no changes to your station information, enter “NOCHANGES” and leave columns G to BB empty.

If you have no stations deployed
If you have no station deployed or being decommissioned, enter “NOSTATIONS” and leave columns G to BB empty

Potential error(s)

The Station Location is missing or exceeds the character limit.

An error in this field may also introduce errors in the following fields:
  • L – Latitude
  • M – Longitude
  • AD – Download resource allocation

G

Station type

Enter the station type of either fixed or earth station.

Example

FX

Format

Must be either:

  • FX - Fixed station
  • TC - Earth station

Potential error(s)

The Station Type is not input as CAPS, is missing or is not an accepted Station Type value.

H

Radio technology

The mode of operation (provisioning) for the radio apparatus at the site.

Example

LTE

Format

  • GSM – Global System for Mobile Communications
  • CDMA – Code Division Multiple Access
  • HSPA – High Speed Packet Access
  • LTE – Long Term Evolution
  • 5GNR – 5G New Radio 
  • 5GDSS – 5G Dynamic Spectrum Sharing
  • WIMAX – Worldwide Interoperability for Microwave Access
  • WI-FI
  • OTHER

Potential error(s)

The Radio Technology value is missing, not in ALL CAPS, or not one of the allowed values.

I

Cell ID

The mobile network Cell Global Identity (CGI) following the 3GPP notation for the given radio technology.

Required when

This field is required when H - Radio technology is any of the following:

  • GSM
  • CDMA
  • HSPA
  • LTE
  • 5GNR
  • 5GDSS

Example

302610-1051200-25381093417

Format

For GSM, CDMA, HSPA
(CGI): PLMN-LAC-CID

For 5GNR, 5GDSS
(NCGI): PLMN-TAC-NCI

For LTE
(ECGI): PLMN-TAC-ECI

  • Must be in the format of 999999-99999999-99999999999
  • Minimum of 11 characters
  • Maximum of 27 characters

Potential error(s)

The Cell ID is missing or not in the expected format or length.

J

Physical cell ID

Enter the Physical Cell Identity (PCI) for LTE or 5G systems.

Required when

This field is required only when H - Radio technology is any of the following:

  • LTE
  • 5GNR
  • 5GDSS

Example

204

Format

Must be a number from either:

  • 0 to 503 for LTE systems
  • 0 to 1007 for 5GNR or 5GDSS systems

Potential error(s)

The Physical Cell ID is missing or not within the expected range of values.

K

Province/Territory code

Enter the two-character abbreviation of the province or territory your station is located in.

Example

ON

Format

Your licence coordinates must be within the province or territory

Must be one of the following provincial or territorial codes:

  • AB - Alberta
  • BC - British Columbia
  • MB - Manitoba
  • NB - New Brunswick
  • NL- Newfoundland and Labrador
  • NS - Nova Scotia
  • ON - Ontario
  • PE - Prince Edward Island
  • QC - Quebec
  • SK - Saskatchewan
  • NT - Northwest Territories
  • NU - Nunavut
  • YT - Yukon
For transportable stations, includes "IP" for interprovincial and "CA" for Canada-wide.)

Potential error(s)

The Province/Territory is not input as CAPS, is missing or is not an accepted Province/Territory abbreviation.

L

Latitude

Enter the latitudinal coordinate of the station.

Example

45.3514035

Format

  • Must be in decimal degrees using the WGS84 format
  • Must be a location within the licence area
  • Must be between 41 to 90
  • A minimum of four decimal places may be entered if coordinates are not exact
  • A maximum of seven decimal places

Potential error(s)

The Latitude is missing or not within the expected range. The site coordinates may not fall within the licensed service area.

M

Longitude

Enter the longitudinal coordinates of the station.

Example

-75.6533946

Format

  • Must be in decimal degrees using the WGS84 format
  • Must be a location within the licence area
  • Must be between -141 to -40
  • A minimum of four decimal places may be entered if coordinates are not exact
  • A maximum of seven decimal places

Potential error(s)

The Longitude is missing or not within the expected range (use negative to indicate degrees west). The site coordinates may not fall within the licensed service area.

N

Site type code

The general type of site associated with the installation.

Example

O

Format

Must be one of the following site types:

  • U - Underground (E.g. a mine or a subway)
  • O - Outdoor (E.g. a tower or rooftop)
  • I - Indoor (E.g. a stadium or office)

Potential error(s)

The Site Type Code is not input as CAPS, is missing or is not an accepted Site Type Code value.

O

Structure height

The total height (m) above ground level of an outdoor structure where antennas are mounted. This structure might also be shared with other radio installations (ex: towers, rooftops, etc).

Required when

This field is required only when N - Site type code is O.

Example

15

Format

  • Must be greater than the antenna height
  • Must be between 1 to 600 m
  • For equipment mounted on top of a building or its boundaries, add together the total height of the building and the antenna’s supporting structure.

Potential error(s)

The Structure Height is missing or not within the expected range of values.

An error in this field may also give errors in the following fields:
  • AM - Tx Antenna Height
  • AN - Rx Antenna Height

P

Site structure type code

The type of structure in use at the site.

Example

T

Format

Must be one of the following structure types:

  • T - Purpose-built tower
  • R - Outside of building or rooftop
  • P - Light pole, other street electrical, or utility pole
  • S - Water tower or farm silo
  • F - Furniture
  • M - Transportable (Report the current operating location only and not the radius of operation)
  • O - Other (E.g. indoor)

Potential error(s)

The Structure Type Code is not input as CAPS, is missing or is not an accepted Structure Type Code value.

Q

Effective date of configuration

The date that the installation went into service. This date helps investigate possible sources of technical issues.

Example

2022-02-14

Format

  • Must be a date in the past
  • Must be in the format of YYYY-MM-DD

Potential error(s)

The date is missing or not in the expected format or range.

ORA-01841: (full) year must be between -4713 and +9999, and not be 0

If you have an improper date format, other errors may not be shown.

R

Site record ID

The identifier as defined by the licensee for the site or antenna installation.

Example

1234567890

Format

(Note that it is not validated to be unique)

Potential error(s)

The Site Record ID is missing.

S

Tx channel frequency

The centre frequency (MHz) of the occupied bandwidth as well as any installations that are used unidirectionally for transmission or reception only.

Example

1925.3

Format

  • If your antenna is receive only and not transmitting enter 0
  • Must be within an authorized frequency band for this licence
  • Must be the centre frequency

Potential error(s)

The Tx Channel Frequency is missing or not within an authorized frequency band for this licence.

T

Rx channel frequency

The centre frequency (MHz) of the occupied bandwidth for the given antenna installation, as well as any installations that are used unidirectionally for transmission or reception only.

Example

1875.3

Format

  • Must be within an authorized frequency band for this licence
  • Must be the centre frequency
  • If your antenna is transmit only and not receiving enter 0

Potential error(s)

The Rx Channel Frequency is missing or not within an authorized frequency band for this licence.

U

Tx radio HVIN

The transmitting radio’s Hardware Version Identification Number (HVIN) certified by ISED.

Required when

This field is required when S - Tx channel frequency is greater than 0.

Example

T99W175MC

Format

  • Maximum length 128 characters
  • If the radio is certification-exempt, you must provide the manufacturer's radio model number or a system configuration name

Potential error(s)

The Tx Radio HVIN is missing or exceeds the character limit.

V

Rx radio HVIN

The receiving radio’s Hardware Version Identification Number (HVIN) certified by ISED. If the radio is certification-exempt, please provide the manufacturer's radio model number or a system configuration name.

Required when

This field is required when T - Rx channel frequency is greater than 0.

Example

T99W175MC

Format

  • Maximum length 128 characters
  • If the radio is certification-exempt, you must provide the manufacturer's radio model number or a system configuration name

Potential error(s)

The Rx Radio HVIN is missing or exceeds the character limit.

W

Tx radio manufacturer code

The company code assigned by ISED’s Certification and Engineering Bureau (CEB) to the manufacturer of the radio transmitter.

Required when

This field is required when G - Station type is FX and S - Tx channel frequency is greater than 0.

Example

109AB

Format

  • Must be a valid manufacturer company number

Potential error(s)

The Tx Radio Manufacturer Code is missing or is not an accepted Manufacturer Code value.

X

Rx radio manufacturer code

The company code assigned by ISED’s Certification and Engineering Bureau (CEB) to the manufacturer of the radio receiver.

Required when

This field is required when G - Station type is FX and T - Rx channel frequency is greater than 0.

Example

109AB

Format

  • Must be a valid manufacturer company number

Potential error(s)

Y

Tx radio certification number

The Certification Number assigned by ISED to the radio transmitter and listed in the Radio Equipment List (REL).

Required when

This field is required when G - Station type is FX and S - Tx channel frequency is greater than 0.

Example

24374-T99W175M

Format

  • Must be a valid certification number from the Radio Equipment List (subservice 12-142)
  • If your equipment is exempt from certification, you must enter "Exempt"
  • Maximum length 31 characters

Potential error(s)

The Tx Radio Certification Number is missing or exceeds the character limit.

Rx radio certification number

AA

Bandwidth

The occupied bandwidth (KHz) of the emission (Transmit) signal. It corresponds to the bandwidth containing 99% of the transmitted power.

Example

2300

Format

  • A number between 0.10 to 50000.00

Potential error(s)

The Bandwidth is missing or not within the expected range.

AB

Class of emission

The International Telecommunication Union (ITU) designation for the class of emission.

Example

F3WET

Format

  • Must be a valid emission code as described in Valid Class of Emission Codes.
  • Enter the emission class that uses the largest bandwidth

Potential error(s)

The Class of Emission is missing or not in the expected format or length.

AC

Transmitter TCP–TRP

The Total Radiated Power (TRP) or Total Conducted Power (TCP) in dBm.

Ensure the value entered is expressed in dBm. Wrong units will cause incorrect representation of power levels, and may be flagged for suspected non-compliance.

Required when

This field is required when S - Tx channel frequency is greater than 0.

Example

21.4

Format

Must be one of the following:

  • Enter TRP for AAS equipment (as defined in the applicable SRSP)
  • Enter TCP for non-AAS equipment

Potential error(s)

The Transmitter TCP or TRP is missing, or not within the expected range.

AD

Downlink resource allocation

The percentage of time transmitting.

Example

89

Format

  • For bidirectional systems that transmit and receive on the same frequency (TDD), enter the percentage of time transmitting (see GL-01) from 1 to 99.
  • For full duplex systems, enter 100.

Potential error(s)

The Downlink Resource Allocation is missing or not within the expected range.

AE

Tx antenna type code

The type of antenna associated with a transmitting radio installation.

Required when

This field is required when G - Station type is FX and S - Tx channel frequency is greater than 0.

Example

NAU

Format

Entry must be one of the following:

  • AAS – Active antenna systems
  • NAC – Non-AAS correlated
  • NAU – Non-ASS uncorrelated

If deploying a hybrid mode, enter NAC.

Potential error(s)

The Tx Antenna Type Code is not input as CAPS, is missing or is not an accepted Antenna Type Code value.

Required when

The total number of Tx antennas.

Required when

This field is required when T - Rx channel frequency is greater than 0.

Example

1

Format

  • For a single-input and single-output (SISO) system enter 1
  • For a traditional antenna diversity enter 2 or any numbes greater
  • For MIMO systems, enter the number of antenna elements in the antenna panel

    • Potential error(s)

    The Number of Tx Antennas is missing or not within the expected range.

Maximum length of 80 characters.

AM

Tx antenna height

Height (m) of physical centre of the transmitter antenna above or below ground level in meters including the structure/building on which the antenna is mounted.

Required when

This field is required when S - Tx channel frequency is greater than 0.

Example

15

Format

If N - Site Type Code is O (Outdoor):
The antenna height must be between 0 to 600 m and no more than 5 m taller than the O - Structure height.

If N - Site Type Code is I (Indoor):
The antenna height must be between 0 to 600 m.

If N - Site Type Code is U (Underground):
The antenna height must be between -100 to 0 m.

Potential error(s)

The Tx Antenna Height is missing or not within the expected range.

AQ

Tx antenna horizontal beam

The beam steering angle for AAS (Active Antenna Systems) or the beam width in the horizontal plane for non-AAS.

Required when

This field is required when S - Tx channel frequency is greater than 0 and AO - Tx antenna omnidirectional pattern indicator is D (Directional).

Example

120

Format

For AAS
Enter the horizontal beam steering angle range around the mean value specified by AU - Tx Antenna Azimuth.

For non-AAS
Enter the half-power (3 dB) beam width in the horizontal plane of the main lobe for the transmitter antenna.

Must be a number from 1.0 to 359.9 (degrees).

Potential error(s)

The Tx Antenna Horizontal Beamwidth is missing or not within the expected range.
