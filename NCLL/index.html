Spectrum Licensing Services

Spectrum licensing is central to the delivery of quality wireless services and networks for use in Canada. On this page, registered licence users will be able to pay invoices, search, browse and apply for spectrum licences, as well as upload and view spectrum licence technical data.

On this page:

  • Pay Online
  • Search or Apply for Spectrum Licences
  • Spectrum Licence Technical Data
  • Non-competitive local licensing (NCLL) licences
 

Pay Online

As radio and spectrum licence holders, you must pay licence fees to keep them current and active. Visit Pay Online for more information.

Search or Apply for Spectrum Licences

Applying for an available spectrum licence:

  1. Create a Web Profile. To find out how to register, see Creating a Web Profile.
  2. Log in to your Web Profile.
  3. Use Spectrum Licences search tool to find Spectrum Licences available for application.
  4. If your Web Profile has multiple associated licensee accounts, use the Select Active Licensee option to select the licensee account that your applications are submitted under.

Searching for spectrum licences

If you are looking for available spectrum licences to apply:

  1. Go to Spectrum Licences available for application.
  2. Select a Licence Category from the dropdown menu.
  3. Click Search.
  4. To see the Licence Data, click the Authorization Number of the desired licence from your search results.

Browsing Spectrum Licences

  1. Go to Spectrum Licence Browser.
  2. Insert search parameters.
  3. Click Search.
  4. To see the Licence Data, click the Authorization Number of the desired licence from your search results.

Spectrum Licence Technical Data

Companies with spectrum licences that operate wireless services in Canada must provide technical data on their authorized stations when they become operational or, technical changes are made. Use the following options to upload or view this data.

  • Spectrum Licence Technical Data Upload
  • Spectrum Licence Technical Data Upload Browser
  • Spectrum Licence Station Browser
  • Requirements for Submission of Technical Data
  • Spectrum Licence Technical Data Download

If you require further information, please contact your nearest ISED District Office.
